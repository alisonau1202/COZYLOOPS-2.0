1. Unzip the zip file called 102768672_COS30043_project_25
2. Make sure you have a folder called COS30043. The folder structure should be as shown below:
   /COS30043
   ├── /project
   ├── ReadMe.txt
3. Place the COS30043 folder in xampp/htdocs.
4. Go to phpMyAdmin.
5. Create database - project (use project as database name)
6. Click on import and import these 4 sql files:
	- addresses.sql
	- cart.sql
	- purchases.sql
	- users.sql

   /COS30043
   ├── /project
	├── /sqls
	│   └──├── addresses.sql
	│       ├── cart.sql
	│       ├── purchases.sql
	│       ├── users.sql


7. Open the link below and start using the website:
	- http://localhost/COS30043/project
